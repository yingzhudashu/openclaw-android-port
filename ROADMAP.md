## 当前状态 vs 桌面端能力差距分析

### ✅ 已实现（基本可用）
| 功能 | 桌面端 | 安卓端 | 差距 |
|------|--------|--------|------|
| 多供应商 LLM | ✅ | ✅ | 基本一致 |
| 会话管理 | ✅ | ✅ | 基本一致 |
| Markdown 渲染 | ✅ | ✅ | 代码高亮待优化 |
| 图片分析 | ✅ | ✅ | 基本一致 |
| 文件分析 | ✅ | ✅ | 基本一致 |

| 双语 UI | ✅ | ✅ | 基本一致 |
| 供应商配置 | ✅ | ✅ | 基本一致 |

### 🔴 关键差距（按优先级排序）

#### P0 - 体验致命缺陷
1. **无流式响应（Streaming）** - 桌面端实时逐字输出，安卓端要等整个回复完成才显示，复杂任务等待时间可达分钟级，用户以为卡死
2. **超时/断连频繁** - 移动网络不稳定 + 长时间同步请求 = 频繁报错
3. **无思考过程展示** - 桌面端显示 thinking/reasoning，安卓端不显示

#### P1 - 核心功能缺失
4. **无工具调用可视化** - Agent 执行工具时用户看不到过程（搜索中/浏览中/计算中）
5. **无记忆系统** - 没有 MEMORY.md、每日日记、长期记忆
6. **Skills 无法执行** - 只能安装/列出，不能真正运行
7. **无状态监控** - 状态页只有基本信息，无引擎日志/健康检查
8. **无消息重发/编辑** - 发送失败没有重试按钮

#### P2 - 体验优化
9. **无代码复制按钮** - 代码块没有一键复制
10. **无消息长按菜单** - 不能复制/删除/重发单条消息
11. **无深色模式** - 只有浅色主题
12. **无网络状态提示** - 断网时没有提示
13. **输入框不能多行** - 不能输入长文本
14. **无图片预览** - 发送/接收的图片没有全屏预览
15. **无语音输入** - 不能语音转文字

#### P3 - 进阶能力
16. **无本地文件浏览** - 不能浏览手机文件系统
17. **无通知推送** - 后台消息没有通知
18. **无快捷指令** - 没有 /commands
19. **无多轮对话上下文管理** - 不能清除/编辑上下文
20. **无离线模式** - 没有本地 LLM 支持

## 迭代开发计划

### Phase 1: 流式响应 ✅
**目标**: 让用户实时看到 AI 回复，彻底解决"卡死"感
- [x] Gateway: `/api/chat` SSE 端点
- [x] Kotlin: SSE 客户端解析 `text/event-stream`
- [x] UI: 逐字追加消息，打字机效果
- [x] 思考过程: 显示 reasoning 折叠区域
- [x] 工具调用: 显示工具调用步骤

### Phase 2: 错误处理与稳定性 ✅
**目标**: 在各种网络条件下都能正常工作
- [x] 网络状态检测与提示（NetworkCallback + 红色横幅）
- [x] 请求重试机制（指数退避 1s→3s，最多 2 次）
- [x] 消息发送失败 → 重发按钮
- [x] Gateway 健康检查（30s 心跳）
- [x] 输入框草稿保存（SharedPreferences）
- [x] ACCESS_NETWORK_STATE 权限

### Phase 3: 消息交互增强 ✅
**目标**: 接近微信/Telegram 级别的消息交互
- [x] 代码块复制按钮（[复制代码] ClickableSpan）
- [x] 消息长按复制（已有 copyToClipboard）
- [x] 多行输入框（textMultiLine + maxLines=5）✅ 已有
- [x] 图片全屏预览（showFullImage Dialog）✅ 已有
- [x] 链接点击跳转（Markdown [text](url) + 裸链接 http://）

### Phase 4: 记忆与上下文 ✅
**目标**: 让 AI 能记住用户，跨会话保持连贯
- [x] Gateway: 记忆文件读写 API（已有 /api/memory）
- [x] UI: MEMORY.md 编辑器（设置页新增 Memory 分区）
- [x] 会话上下文管理（会话菜单“清除上下文”）
- [x] 系统提示词编辑（SOUL.md 已有）
- [ ] 记忆搜索（待 Gateway 支持）

### Phase 5: 高级功能
**目标**: 接近桌面端完整能力
- [ ] Skills 执行引擎
- [ ] 状态监控仪表盘
- [ ] 深色模式
- [ ] 通知推送
- [ ] 语音输入
- [ ] 快捷指令
